Useful resources

Development

APIs

  • Swagger editor: quickly check openapi files
  • Postman: test APIs with a better UX than Swagger (imo)

Inspect HTTP requests:

These allow you to inspect your HTTP requests, such as POST:

  • ngrok: secure introspectable tunnels to localhost
  • Webhook.site: saves requests and lets you inspect them
  • httpbin(1): test requests. Simple, but provides a docker image to use locally. Try it with docker run -p 80:80 kennethreitz/httpbin

Others

  • JSON formatter: validate json online, although I prefer JSON Reindent for sublime text
  • Regex 101: learn and validate regex

Music

  • Who Sampled: discover the samples used in a song, or songs which sampled it

Learning

Anki is a website which lets you generate and review flash cards.

Online Hosting

neocities is a free website hosting service with a basic, but neat interface.
