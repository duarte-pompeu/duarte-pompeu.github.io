Useful resources

Below are some resources which I find very useful and frequently access.

Development

APIs

  • Swagger editor: inspect and edit openapi files
  • Postman: test APIs with a scriptable interface and UX slightly better than Swagger

Inspect HTTP requests:

These allow you to inspect your HTTP requests, such as POST:

  • ngrok: secure introspectable tunnels to localhost
  • Webhook.site: saves requests and lets you inspect them
  • httpbin(1): test requests. Simple, but provides a docker image to use locally. Try it with docker run -p 80:80 kennethreitz/httpbin

Others

  • Regex 101: learn and validate regex
  • Crontab Guru: validate cron online

Arts

  • Who Sampled: discover the samples used in a song, or songs which sampled it
  • Every noise at once: map of music genres with samples
  • tastedive: find similar things to your favorite things, ie books/games/movies/music/tv shows
  • Steam tools - item value: load your inventory and sort the items by market value
  • Steam shark: search discounts for games

Typing

Quick typing is a hobby of mine, and my personal record was 115 WPM (words per minute).

This was back when I was using 3 fingers on each hand. Now I’m trying the proper style using all fingers, and I’m able to reach around 90 WPM. Decades olds habbits are pretty hard to break!

  • 10 fast fingers
  • keybr
  • The Practice Test

Learning

Anki is a website which lets you generate and review flash cards.

You Pronounce It let’s you find the proper pronunciation of words by searching youtube videos.

Online Hosting

netlifty is what I currently use to build my websites.

000webhost is a free server, with small space available and only a few tools (php, mysql, cron and ftp)… but it’s free!

neocities is a free website hosting service with a basic, but neat interface.

Others

  • Have I been Pwned: check if you have an account that has been compromised
  • Bitwarden Send: share passwords securely (password protection + expiry date + deletion after n accesses)
  • Untools: tools for problem-solving with diagrams
  • Start Bodyweight: exercises for bodyweight training
  • Gordon Brander’s website: interesting essays
    • Architects vs gardeners
  • Without bullshit: blog about technical writing
  • Guerilla mail: temporary email to register in crap sites so they won’t spam you
